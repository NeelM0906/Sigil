# `sigil setup`

Initialize `~/.sigil/sigil.json` and the agent workspace.

Related:
- Getting started: [Getting started](/start/getting-started)
- Wizard: [Onboarding](/start/onboarding)

## Examples

```bash
sigil setup
sigil setup --workspace ~/sigil
```

To run the wizard via setup:

```bash
sigil setup --wizard
```
