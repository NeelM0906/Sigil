# `sigil tui`

Open the terminal UI connected to the Gateway.

Related:
- TUI guide: [TUI](/tui)

## Examples

```bash
sigil tui
sigil tui --url ws://127.0.0.1:18789 --token <token>
sigil tui --session main --deliver
```
